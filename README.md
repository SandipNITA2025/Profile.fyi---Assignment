## Features

- Shopping cart functionality
- Discount code application
- Responsive design

## Tech Stack

- Next.js
- TypeScript
- Redux Toolkit
- Tailwind CSS

## Installation

To install the dependencies, run the following command:

```bash
npm install
```

## Running the Application Locally

To start the application on your local machine, run:

```bash
npm run dev
```
